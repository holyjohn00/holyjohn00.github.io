1. Content-Based Filtering

The Scenario

We want to know if we should recommend the movie "Inception" to User A.


User A Profile: Likes Action and Sci-Fi.

Item (Inception): Genres are Action and Sci-Fi.

Item (Notebook): Genre is Romance.

The Data Vectors

We convert genres into binary vectors [Action, Sci-Fi, Romance].


EntityActionSci-FiRomanceVector
User A Profile110A = [1, 1, 0]
Movie: Inception110B = [1, 1, 0]
Movie: Notebook001C = [0, 0, 1]

Content-Based: The Calculation

We use Cosine Similarity to measure the angle between the User Vector and the Item Vector.

Formula

Sim(A, B) = (A . B) / (||A|| * ||B||)


Step 1: Dot Product (A . B)

(1*1) + (1*1) + (0*0) = 2


Step 2: Magnitude (||A||)

√(1² + 1² + 0²) = √2 = 1.414

Solution

Target: Inception [1, 1, 0]
Numerator: 2
Denominator: 1.414 * 1.414 = 2
Similarity = 2 / 2 = 1.0 (Perfect Match)
Target: Notebook [0, 0, 1]
Dot Product: (1*0) + (1*0) + (0*1) = 0
Similarity = 0 / 2 = 0.0 (No Match)
Result: System recommends Inception.

2. User-Based Collaborative Filtering

The Scenario

Goal: Predict the rating User A would give to Item 3.

We look for other users (neighbors) who are similar to User A.

User-Item Matrix

UserItem 1Item 2Item 3 (Target)Item 4
User A45?2
User B4551
User C1125

User-Based: The Calculation

Step 1: Similarity

Calculate Cosine Similarity between User A and others based on common items (Item 1, 2, 4).


A = [4, 5, 2]

B = [4, 5, 1]

C = [1, 1, 5]


Sim(A, B): High (Similar ratings)

Sim(A, C): Low (Opposite ratings)

(Let's assume Sim(A,B) = 0.9 and Sim(A,C) = 0.1 for simplicity)

Step 2: Weighted Prediction

Formula: Σ(Similarity * Rating) / Σ(Similarity)

User B's influence:
Sim(0.9) * Rating(5) = 4.5
User C's influence:
Sim(0.1) * Rating(2) = 0.2
Total: (4.5 + 0.2) / (0.9 + 0.1)
= 4.7 / 1.0 = 4.7
Prediction: User A will rate Item 3: 4.7 Stars

3. Item-Based Collaborative Filtering

The Scenario

Goal: Predict rating for User A on Pizza.

Instead of looking at users, we look at how similar Pizza is to items User A has already rated (e.g., Burger).

User A rated Burger: 5 Stars.

Item-Item Matrix (Column Vectors)

UserBurgerPizza (Target)Salad
User X551
User Y452
User Z115

Notice: Burger and Pizza have very similar columns. Salad is different.

Item-Based: The Calculation

Step 1: Item Similarity

We treat the Items as vectors (columns from previous slide).


Burger Vector = [5, 4, 1]
Pizza Vector = [5, 5, 1]


Dot Product: (25 + 20 + 1) = 46

Magnitudes: ||B||=6.48, ||P||=7.14

Similarity: 46 / (6.48 * 7.14) ≈ 0.99

Step 2: Prediction

Since Pizza is 99% similar to Burger, and User A rated Burger 5 stars:

Prediction = Sim(Burger, Pizza) * UserA_Rating(Burger)
Calculation = 0.99 * 5 = 4.95
Solution: We recommend Pizza to User A (Predicted 4.95 Stars).
